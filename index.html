About

Fuq Docs is a free as in beer to use utility for locally editing, managing, and previewing notes; specifically in Markdown. Fuq Docs has zero: profile-tracking, ad-serving, data collection, data sharing, et al.

Have a question about Fuq Docs or experiencing an issue? Feel free to open an issue in the GitHub repo.

Fuq Docs is a product-of and maintained-by Gus Knowlden.
